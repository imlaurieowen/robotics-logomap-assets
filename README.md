# robotics-logomap-assets
This repository contains all company logos for the European Robotics Landscape Map, organized by layer and category.

Usage
For Claude (AI) Automation
Fetch the full logo index in one call using this URL:
https://raw.githubusercontent.com/Obafunto/robotics-logomap-assets/main/logos.json

The logos.json file contains all 167 companies mapped to their logo URLs, organized by layer and category.

Direct Logo Access
https://raw.githubusercontent.com/Obafunto/robotics-logomap-assets/main/[Layer]/[Category]/[company-name].[ext]
Total Coverage

167 companies across 18 categories
Layer A — 12 application-layer categories
Layer B — 6 infrastructure-layer categories
